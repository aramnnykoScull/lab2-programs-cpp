# Лабораторная работа №2: Линейные программы на C++

**Цель работы**: Изучить разработку и реализацию программ линейной структуры, освоить базовые операции ввода-вывода и простые вычисления.

## Задание
1. Вычисление площади квадрата по введённой стороне.
2. Вычисление объёма прямоугольного параллелепипеда по трём сторонам.
3. Вычисление пройденного расстояния по скорости (км/ч) и времени (ч).

Все программы написаны на C++ и используют общую функцию безопасного ввода `read_int`, которая проверяет корректность ввода (положительное число).

## Требования
- Компилятор с поддержкой C++17 (например, `g++` или `clang++`).
- Утилита `make`.
- (Опционально) Для сборки отчёта — `xelatex` (входит в дистрибутивы TeX Live, MikTeX).

## Сборка и запуск
Все команды выполняются из корневой папки проекта.

### Сборка всех программ
```bash
make all
```
Исполняемые файлы `point1`, `point2`, `point3` появятся в каталоге `bin/`.

### Запуск конкретной программы
```bash
make run_point1   # площадь квадрата
make run_point2   # объём параллелепипеда
make run_point3   # расстояние
```

### Запуск всех программ последовательно
```bash
make run_all
```

### Очистка сгенерированных файлов
```bash
make clean
```

### Полная пересборка
```bash
make rebuild
```

### (Опционально) Сборка отчёта в PDF
```bash
make report
```
Для этого требуется установленный XeLaTeX. PDF будет создан в папке `report/` как `lab2.pdf`.
